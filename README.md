# LVM Practice Project

This repository contains my LVM (Logical Volume Manager) practice using loop devices in Linux.

I created this project to understand how LVM works without using real hard disks.  
Disk image files are used and attached as loop devices.

---

## What I Learned
- Basics of LVM
- Physical Volume (PV)
- Volume Group (VG)
- Logical Volume (LV)
- Formatting and mounting LVM volumes

---

## What This Project Does
- Creates disk image files
- Attaches them as loop devices
- Creates Physical Volumes (PV)
- Creates a Volume Group (VG)
- Creates a Logical Volume (LV)
- Formats and mounts the Logical Volume
